# Support-Ticket-System

A web application that let the user send their enquiry to the moderators of the website. Moderators can reply to the user and mark the question with "solved", "non solved"
. The user will receive an email when moderators send a response.


## INSTALLION :

### in `frontend` folder, type `npm install` to install all dependencies listed in `package.json`, python has the venv folder ready for you, you dont need to install any backend dependency.

### generate a secret key for backend and put inside settings.py and store it in a variable, call it `SECRET_KEY`, to generate a random key: https://djecrety.ir/
